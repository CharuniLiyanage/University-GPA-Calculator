Project Overview

The Student Performance Analysis System is a data analytics project designed to help students monitor and improve their academic performance. Using your GPA and subject grade points, the system identifies weak subjects, tracks GPA trends, and provides personalized suggestions for improvement.

Project Aim

To analyze student academic performance using grade data in order to:

  • Identify weak subjects
  • Observe performance trends across levels and semesters
  • Provide improvement suggestions based on the student's own data

Project Objectives

  • Analyze subject-wise grade points of the logged-in student
  • Calculate and visualize GPA trends per level and semester
  • Detect subjects needing improvement
  • Provide personalized, actionable academic suggestions

Tools & Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Data Handling: JavaScript objects (or backend/CSV)
  • Visualization: Chart.js for GPA & subject trends
  • Optional Backend: Python, Pandas, or SQL database for storing grades

Target Users

  • University students who want to track their GPA and subject performance
  • Academic advisors who want to give personalized guidance
